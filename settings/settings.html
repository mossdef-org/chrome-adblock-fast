How often to check the service status (10-240 seconds)

Setup Instructions

  1. Install luci-app-adblock-fast on your OpenWrt router
  2. Open LuCI: Services → AdBlock-Fast → Advanced Configuration
  3. Copy the Remote Access Token and paste it above
  4. If using HTTPS with a self-signed certificate, visit your router URL in a browser tab first and accept the certificate
